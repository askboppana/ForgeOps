# How to Create a Pull Request

## From ForgeOps Dashboard
1. Go to Pull Requests page → Create PR tab
2. Select your repository from the dropdown
3. Select source branch (your feature branch)
4. Select target branch (int for developers, qa/stage/main for release engineers)
5. Enter Jira ticket number
6. Enter title and description
7. Click "Create Pull Request"

## From VS Code
1. Install GitHub Pull Requests extension
2. Click the PR icon in sidebar
3. Click "Create Pull Request"
4. Fill in details → Create
